SFK KindTrack - Phone Install / PWA Update

Added files:
- manifest.json
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png

Important:
- Upload ALL files and folders to GitHub Pages, including the icons folder.
- PWA install works best when opened from the hosted HTTPS URL, not from file:///Downloads.
- After upload, open the live link on phone.

Android Chrome / Edge:
1. Open the live SFK KindTrack link.
2. Tap the browser menu (⋮).
3. Tap Install app or Add to Home screen.
4. Open KindTrack from the new phone icon.

iPhone:
1. Open the live SFK KindTrack link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Open KindTrack from the new phone icon.

If the old version still appears:
- Refresh the app once or clear site data/cache, then reopen.
